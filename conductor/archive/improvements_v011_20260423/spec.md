# Track Specification: Improvements v0.1.1

## Overview
This track implements the features defined in the v0.1.1 Epic, focusing on improving the user experience, collaboration features, and session management for the planning poker application.

## Features
1. **Card Management:**
   - Users can add cards manually.
   - Users can edit existing cards.
2. **Countdown for Reveal:**
   - Visual countdown sequence triggered after selecting "reveal score".
3. **Session Persistence:**
   - User sessions and room context are maintained across page reloads or accidental navigation.
4. **Admin Transfer:**
   - Current admin can transfer administration rights to another user.
   - Target user must accept or decline the transfer.
5. **Direct Card Selection:**
   - Admin can select any card directly from the list for voting, skipping sequential navigation.
