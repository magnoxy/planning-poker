# Initial Concept
A planning poker application with real-time collaboration using React and Socket.IO.

# Product Guide

## Target Users
- **Agile Software Teams:** Designed for Scrum masters, developers, and product owners to collaborate efficiently during sprint planning meetings.

## Core Features
- **Custom Voting Scales:** Support for various estimation scales, such as Fibonacci or T-Shirt sizing, to adapt to team preferences.

## Value Proposition
- **Streamlined Planning:** Making sprint planning meetings faster, more focused, and more efficient.
