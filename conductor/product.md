# Initial Concept
A planning poker application with real-time collaboration using React and Socket.IO.

# Product Guide

## Target Users
- **Agile Software Teams:** Designed for Scrum masters, developers, and product owners to collaborate efficiently during sprint planning meetings.

## Core Features
- **Manual Task Management:** Add, edit, remove, and reorder tasks (via drag-and-drop) directly in the app.
- **Session Persistence:** State is maintained across page reloads and accidental disconnections.
- **Admin Control:** Secure room management with the ability to transfer administration rights to other participants.
- **Custom Voting Scales:** Support for various estimation scales, with consensus calculated as a Fibonacci-rounded average.
- **Visual Countdown:** Engaging full-screen countdown sequence before revealing votes.

## Value Proposition
- **Streamlined Planning:** Making sprint planning meetings faster, more focused, and more efficient.
